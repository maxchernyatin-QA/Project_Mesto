В рамках проекта была поставлена цель выполнить регрессионное тестирование веб-приложения "Mesto" по готовым тест-кейсам.

Задачи:

1. Познакомиться с сервисом Mesto.

2. Протестировать сервис по тест-кейсам и проставить статусы выполнения проверок.

3. Перепроверить баги, то есть провести ретест.

4. Составить баг-репорты.

Выполнено:

1. Протестировано 17 готовых тест-кейсов, выявлено 9 багов.

2. Перепроверено 15 багов, из которых 10 закрыты, 1 заблокирован, 4 переоткрыты.

3. Составлено 13 баг-репортов.
