В проекте была поставлена цель выполнить регрессионное тестирование веб-приложения "Mesto" по готовым тест-кейсам.

Задачи:

1. Познакомиться с сервисом Mesto.

2. Протестировать сервис по тест-кейсам и проставить статусы выполнения проверок.

3. Перепроверить баги, то есть провести ретест.

4. Составить баг-репорты.
